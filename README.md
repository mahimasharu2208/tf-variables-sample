# tf-variables-sample

This repository demonstrates the use of input variables in Terraform.

## Files
- `main.tf` – Uses variables for configuration.
- `outputs.tf` - Outputs the defined values.
- `variables.tf` – Declares the input variables.

## Usage

```bash
terraform init
terraform apply
```
